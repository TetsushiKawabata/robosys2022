# robosys2022
本リポジトリはロボットシステム学2022の練習のためのものである

# plusコマンド
![test](https://github.com/TetsushiKawabata/robosys2022/actions/workflows/test.yml/badge.svg)

標準入力から読み込んだ数字を足す。
```sh
$ seq 5 | ./plus
15
```

## 必要なソフトウェア
* Python 3.7 ~ 3.10 (テスト済み)

## テスト環境
* ubuntu18.04
